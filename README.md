## サービス概要
フクロモモンガに特化した健康管理が出来るサービスです。  
ケージの気温・活動時間・夜鳴きの有無など、一般的なペットアプリでは管理できない項目を記録できます。  
日々の変化を記録し可視化することで、体調不良や異変に気づきやすくなります。


## このサービスへの思い・作りたい理由
フクロモモンガは体調不良を表に出しにくく、異変に気づいたときには症状が進行していることも少なくありません。現在フクロモモンガを飼育しており、まもなく高齢期に入るため、これまで以上に健康管理の重要性を感じています。

しかし、フクロモモンガは飼育している人が少ないマイナーな動物ということもあり、一般的なペット管理アプリでは対応しておらず、フクロモモンガの健康管理にとって重要な「ケージの気温・活動時間・夜鳴きの有無」などの項目を記録することが出来ません。

そのため、それら項目を記録できるようにして日々の小さな変化を見やすくすることで、体調の異変に早く気づける環境を整えたいと思い、このサービスを作ろうと考えました。


## ユーザー層について
- フクロモモンガを飼育している人


## サービスの利用イメージ
- 毎日、ケージの気温・活動時間・夜鳴きの有無などを簡単に記録する
- 記録が一覧・履歴として残り、「いつから変化があったか」を振り返ることができる
- 通院時には、過去の記録を見ながら状況を整理できる


## サービスの差別化ポイント・推しポイント
### 差別化ポイント
フクロモモンガ専用の健康管理アプリであること。  

一般的なペット管理アプリは犬・猫向けの項目（散歩・ワクチン・シャンプー etc.）が中心のため、フクロモモンガの健康管理にとって重要な項目が用意されておらず、メモで代替する必要があります。  
本サービスはこれらを最初から入力項目として用意することで、「どこに書けばいいか」「何を記録すべきか」で迷いにくく、記録することができます。


## 機能候補
### MVPリリース
- ユーザ登録 / ログイン
- フクロモモンガの基本情報登録
  - 名前・性別・年齢・体重
- 記録機能
  - ケージ内の気温・湿度
  - 活動時間（夜実際に動いた時間）
  - 食事管理（摂取量・おやつ・水分量が適切か）
  - 排泄管理（量・色・形に異常が見られないか）
  - 夜鳴き・脱毛・耳の乾燥・自傷行為の有無
  - 気づいたことのメモ
- 記録の検索機能

### 本リリース
- グラフ表示（異変に気づきやすいように各項目ごとにグラフで可視化する）
- 写真・動画のアップロード
- 各指ごとの爪切りの記録
- 通院などのイベント管理
- 複数匹管理機能
- 異変の疑いがある場合の自動判定機能
- 通知機能（記録忘れの防止）
